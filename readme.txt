A plugin allows for custom avatars. In the backend through the user-edit form, in the frontend through a shortcode.

== Description ==

This plugin allows for custom avatars. In the backend, a custom avatar can be uploaded for a user in the user-edit form. In the frontend, an upload form, for the current user, can be shown by using the shortcode [mb-simple-user-avatar].

== Installation ==

1. **Important** Backup your files and database.

2. **(Recommended)** Install directly from the WP Directory through your admin panel Plugins > Add New > Search for "MB Simple User Avatar". Click install under the name, and activate when prompted. 

3. Activate the plugin through the 'Plugins' menu in WordPress (under plugin name: **"MB Simple User Avatar"**).
